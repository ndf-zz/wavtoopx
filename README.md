wavtoopx
========

Convert audio file to OP1/OPZ sample or synth file.

Copyright (C) 2019 Nathan Fraser

REQUIREMENTS:
-------------

- Python \>= 3
- soundfile

INSTALLATION:
-------------

- pip3 install wavtoopx

USAGE:
------

  wavtoopx input.wav output.aif
